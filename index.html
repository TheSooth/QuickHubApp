QuickHub is a tiny but useful OS X application for GitHub lovers. It helps you to quickly access to GitHub directly from your status bar: Repositories, issues, gists and organizations are just one click away. QuickHub will also notify you when something changes on your GitHub artifacts.

QuickHub is available on the Mac App store for just $0.99 / 0.79€
